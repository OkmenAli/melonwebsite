# Melon Website

Public website for Melon at `themelon.app`.

## Stack

- Next.js App Router
- React
- TypeScript
- Plain CSS

## Local Development

```bash
npm install
npm run dev
```

The development server runs at `http://localhost:3000` by default.

## Production Checks

```bash
npm run typecheck
npm run build
```

## Pages

- `/` - public launch homepage
- `/privacy` - privacy policy
- `/terms` - terms of use
- `/support` - App Store / Google Play support contact page

## Domain

The site metadata, sitemap, and robots file are configured for:

```text
https://themelon.app
```

When hosting is ready, point `themelon.app` to the deployment provider and update any provider-specific DNS records.
